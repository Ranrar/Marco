# Astro Theme Test Document

This document demonstrates the **Astro** theme for Marco Markdown Editor, inspired by space and cosmic elements.

## Color Palette

The astro theme features these distinctive colors:
- **Primary Accent**: #a855f7 (Purple)
- **Secondary Accent**: #ec4899 (Pink) 
- **Info Green**: #10b981 (Emerald)
- **Warning Yellow**: #f59e0b (Amber)

## Typography

All headings use the **Inter** font with gradient text effects, while code uses **Monaco** for a clean, space-age appearance.

### Code Examples

Here's some inline `code` with syntax highlighting:

```rust
fn main() {
    println!("Welcome to the Astro theme!");
    let cosmic_gradient = LinearGradient::new(45deg, "#a855f7", "#ec4899");
    apply_astro_glow(&cosmic_gradient);

```

